PackSight (Standalone)

Welcome to PackSight (Standalone)! PackSight is a free tool to help coaches, players, and officials with analyzing each game after it happens.

The version you are using is the standalone version, which means there is no server backend associated with it. Any data you load into this webpage never leaves your computer. The web application version, which has an impressive number of additional features, is currently being built and slated for general availability in Q4 2026!

To get started, you'll need to download the PackSight CRG conversion script, available here.

./packsight-convert <input-file-path> <output-file-path>
    # as an example...
./packsight-convert game1.json game1-packsight.json

Once you've converted your game file, use the file input below to load it into the webpage.
